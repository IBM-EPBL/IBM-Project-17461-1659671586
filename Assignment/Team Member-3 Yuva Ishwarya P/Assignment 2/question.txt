Why is python more popular programming language ?
 Python language is  one of the most accessible programming language with simple syntax.
 It is a dynamically typed programming language,it supports object oriented concepts.
 It is a general purpose and interpreted language.
 Python can be used on a server to create web applications.
 Python can be used alongside software to create workflows.
 Python can connect to database systems. It can also read and modify files.
 Python can be used to handle big data and perform complex mathematics.
 Python can be used for rapid prototyping, or for production-ready software development.

What is the fullform of WSGI?
 WSGI-web Server Gateway Interface
 WSGI plays a vital role at the time when you deploy your Django or Flask application.
 It explains how a web server communicates with python web applications/frameworks and 
 how web applications/frameworks can be chained for processing a request.
